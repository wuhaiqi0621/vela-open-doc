# 设备型号命名规律

::: info 部分内容来自
https://github.com/aurysian-yan/get-mi-watchface
:::

| 首字母           | 数字第一位   | 数字第二位              |
| ---------------- | ------------ | ----------------------- |
| 代表设备发布年份 | 代表设备类型 | 代表设备系列            |
| M(2023)          | 6(可穿戴)    | 1(全智能)               |
| N(2024)          |              | 2(Xiaomi Watch S 系列)  |
| O(2025)          |              | 5(REDMI Watch)          |
| P(2026)          |              | 6(小米手环标准版 / NFC) |
|                  |              | 7(小米手环 Pro)         |
| 以此类推...      |              | 9(红米手环)             |

第二位部分数字仅在一代产品上出现过，无法确定数字是否与产品定位绑定。

- N60：小米腕部心电血压记录仪

- O63：Xiaomi Watch S4 41mm
