# 设备型号命名规律

::: info 部分内容来自
https://github.com/aurysian-yan/get-mi-watchface
:::

## 首字母

代表设备发布年份

- M(2023)
- N(2024)
- O(2025)
- P(2026)
- ...

## 数字第一位

代表设备类型

- 6(可穿戴)

## 数字第二位

代表设备系列

- 1(全智能)
- 2(小米手表 S)
- 5(红米手表)
- 6(小米手环 / NFC)
- 7(小米手环 Pro)
- 9(红米手环)

::: info
第二位部分数字仅在一代产品上出现过，无法确定数字是否与产品定位绑定。

- N60：小米腕部心电血压记录仪
- O63：Xiaomi Watch S4 41mm
:::
